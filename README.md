**Minecraft Admin Panel** — это высокопроизводительная десктопная утилита, предназначенная для комплексной автоматизации задач администрирования серверов Minecraft. Программное обеспечение разработано на языке Python с применением современного фреймворка **Flet** (на базе Flutter), что гарантирует высокую скорость отклика и бесперебойную работу интерфейса.

## 🚀 Ключевые возможности
- **Интеграция с серверами:** Автоматизированное подключение по протоколу SFTP и синхронизация модификаций (удаление устаревших версий, загрузка обновлений) в один клик.
- **RCON Консоль:** Прямое удаленное управление сервером посредством встроенной RCON-консоли. Включает поддержку истории команд, функцию быстрого ввода и подсветку синтаксиса системных логов.
- **Управление резервными копиями:** Автоматизированное создание локальных архивов (на базе алгоритмов 7-Zip) с возможностью исключения нецелевых директорий (например, `world` или `logs`).
- **Сверхкомпактная сборка:** Готовые бинарные файлы (формат EXE для Windows и исполняемый файл для Linux) оптимизированы и сжаты при помощи упаковщика UPX для максимального снижения объема занимаемого пространства.
- **Современный пользовательский интерфейс:** Интегрирована темная тема оформления (Dark Mode), акцентные визуальные элементы и плавная анимация переходов для повышения эргономики работы.
- **Встроенная система обновлений:** Приложение осуществляет автоматическую проверку наличия новых версий в репозитории GitHub и предлагает инсталляцию обновлений без необходимости ручной загрузки.

## 📥 Установка и запуск

Для начала работы необходимо перейти в раздел [Releases](https://github.com/milkycloud-dev/admin-panel-minecraft/releases) и скачать актуальную версию программного обеспечения для используемой операционной системы. 
Продукт поставляется в формате единого исполняемого файла (Portable-версия), что полностью исключает необходимость установки.

## ⚙️ Стек технологий
- **Python 3.11+**
- **Flet** (Современный UI Framework)
- **Paramiko** (Интеграция протоколов SFTP/SSH)
- **PyInstaller & UPX** (Сборка и компрессия исполняемых файлов)
